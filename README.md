## Opis projekta
Ovaj projekt izrađen je u sklopu kolegija **Sigurnost informacijskih sustava (SIS)**.  
Cilj projekta je implementirati i testirati **Web Application Firewall (WAF)** kako bi se razumjelo njegovo djelovanje, konfiguracija pravila te učinkovitost u zaštiti web aplikacija od najčešćih napada na aplikacijskom sloju.

Projekt uključuje teorijski dio (opis WAF-a, arhitektura, mehanizmi detekcije) te praktični dio u kojem se provodi konfiguracija WAF-a i simulacija napada.

---

## Ciljevi projekta
- Implementirati i konfigurirati Web Application Firewall
- Analizirati mehanizme detekcije napada (signature-based, anomaly-based)
- Testirati učinkovitost WAF-a protiv napada poput SQL Injection i XSS
- Analizirati rezultate i identificirati prednosti i ograničenja WAF-a
